# ProtocolosVotacaoQuantica
Este repositório contém os códigos-fonte utilizados para simulação dos protocolos de votação eletrônica quânticos em ambiente Python, com utilização da biblioteca Qiskit, realizados para a apresentação de resultados no artigo "Os Impactos da Computação Quântica na Segurança dos Sistemas de Votação Eletrônica.
